Pré-installation

02. Démarrage via PXE

Accès au serveur PXE

Accès au serveur de déploiement via l'adresse IP 10.248.45.32

03. Début de l'installation

Démarrage du script d'installation Windows

Choix de l'image système à installer sur l'ordinateur.

04. Unité organisationnelle Active Directory

Choix de l'organisation

Sélection de l'arborescence Active Directory dans laquelle l'ordinateur sera référencé. Avant-dernière phase de la pré-installation.

05. Fin de la pré-installation

Nommage de l'ordinateur

Le nom respecte la nomenclature suivante : 'MB' suivi de l'identifiant de l'ordinateur et se termine par 'S'.
Exemple : MB141246S

06. Script d'installation Intune

Configuration Cloud & Logiciels

Intune prend le relais pour appliquer l'image Windows personnalisée. L'OS est déployé avec les logiciels sélectionnés par l'entreprise, tandis que d'autres accès sont restreints.
Exemple : Le Microsoft Store est bloqué et remplacé par le 'Portail d'entreprise'.

07. Fin du déploiement

Clôture de l'installation

L'image Windows est entièrement déployée, la machine est prête pour l'utilisateur final.
